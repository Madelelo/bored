Bored API Documentation

Base URL

https://vercel.com/elvebakken/bored-backend/api

Endpoints

GET /randomactivity

Returns a random activity.

Response

{
  "activity": "Learn a new programming language",
  "type": "education",
  "participants": 1,
  "price": 0,
  "link": "",
  "key": "3943506",
  "accessibility": 0.1
}

GET /activity?type=:type

Returns a random activity of the specified type.

Parameters

  • type (string): The type of activity (e.g., 'education', 'recreational').

Response

{
  "activity": "Write a short story",
  "type": "recreational",
  "participants": 1,
  "price": 0,
  "link": "",
  "key": "1234567",
  "accessibility": 0.1
}

POST /activity

Adds an activity to the database.
